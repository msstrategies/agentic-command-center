## Self-Annealing (Non-Negotiable)

Every error is a lesson the skill learns permanently. When something breaks:

1. Fix the immediate problem
2. Patch the skill so the same failure is impossible next time
3. Resume

**Patch:** wrong values, missing steps, flawed script logic, incorrect assumptions.
**Don't patch:** one-off environmental issues (timeout, rate limit).

Skills get more reliable with each run, converging toward zero-failure execution.
